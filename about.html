Transactional Business Technologies

REST API

Вся функциональность платформы сосредоточена в облаке и рапределена между разными дата-центрами.

Взаимодействие со всеми внешними системами и приложениями осуществляется по унифицированному REST API поверх защищенных HTTP соединений, что позволяет добиться прозрачной балансировки нагрузки, единообразного интерфейса, отказоустойчивости и простоты интеграции.


HTTP-API для приложений

Для интеграции существующих мобильных приложений бизнес-клиентов предусмотрен отдельный облегченный API и наборы кодов компонент для iOS и Android.

Все компоненты и методы доступа к системе оптимизированы по интерфейсам и протоколам, чтобы обеспечить максимальную простоту взаимодействия и максимальную гибкость при использовании (внешний вид, поведение, пользовательские предпочтения), минимально влияя на существующий интерфейс приложения, его поведение и логику взаимодействия с конечным пользователем.
